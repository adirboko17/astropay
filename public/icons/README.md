# אייקון להתקנה על שולחן העבודה (PWA)

## בחירת אייקון משלך

1. הכן תמונה **מרובעת** (למשל 512×512 או 1024×1024), פורמט **PNG** עם רקע (לא חובה שקוף).
2. שמור אותה כאן בשם: **`icon-source.png`**
3. מהשורש של הפרויקט הרץ:

   ```bash
   npm run icons
   ```

4. הסקריפט ייצור:
   - `icon-192.png` — לטלפונים ולמטא-דאטה
   - `icon-512.png` — להתקנה על שולחן העבודה (Windows / Chrome / Edge)
   - `../src/app/icon.png` — ללשונית בדפדפן

5. **פרודקשן:** אחרי שינוי אייקון — `npm run build` (או deploy מחדש).
6. **מחשב שכבר התקין:** לפעמים צריך להסיר את האפליקות הישנה ולהתקין שוב, או לנקות cache, כדי לראות אייקון חדש.

## בלי סקריפט

אפשר להחליף ידנית את `icon-192.png` ו-`icon-512.png` (בדיוק בגדלים האלה).
